# Timetable Generator

A simple, responsive web app to create and customize weekly timetables. Built for GitHub Pages using HTML, CSS, and Vanilla JavaScript.

---

## Features

- Weekly timetable grid (Mon–Fri, coustum hourly slots)
- Add and edit subjects
- Custom subject colors
- Click-to-edit interface
- Auto-save using `localStorage`
- Export timetable as PNG using `html2canvas`
- Clean, modern UI with a blue/white design

---

## languages

- HTML5
- CSS3 (with custom styling + optional Tailwind CDN)
- Vanilla JavaScript
- html2canvas (for image export)

---

## How to Use

1. Click an empty timetable cell to add a subject
2. Enter subject name and choose a color
3. Click an existing subject to edit it
4. Use **Export PNG** to download your timetable image
5. Data is automatically saved in your browser
